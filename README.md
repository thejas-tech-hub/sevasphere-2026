## 🌟 Problem Statement

Every year, 50+ million Indians are affected by natural disasters. Relief organizations lose **72 critical hours** coordinating volunteers. Existing platforms are reactive — they act after disaster peaks, not before. Field data from paper surveys gets lost, never digitized.

## 🚀 Solution

SevaSphere uses **Gemini AI + real UN disaster data (GDACS)** to detect emerging crises and pre-mobilize verified volunteers — connecting NGOs, communities, and first responders in real-time.

**Key differentiator:** We act *before* the crisis peaks, not after.

## ✨ Core Features

| Feature | Description |
|---|---|
| 🌍 **Live Crisis Detection** | Real disaster data from GDACS (UN system) + Gemini AI analysis |
| 📄 **Multimodal Field Reports** | NGOs photograph handwritten surveys — Gemini AI reads and digitizes instantly |
| 🗺️ **Crisis Intelligence Map** | Google Maps with crisis pins, pulse zones, risk scores |
| 🤖 **AI Volunteer Matching** | Gemini matches volunteers by skill, location, and urgency |
| 📋 **AI Situation Report** | One-click government-ready SITREP generated by Gemini AI |
| 📊 **Impact Tracker** | AI-powered volunteer impact scoring with badges |
| 🏢 **NGO Command Center** | Real-time dashboard with crisis alerts and volunteer management |

## 🛠️ Tech Stack

- **Frontend:** React.js, TailwindCSS, Recharts, React Markdown
- **Backend:** Firebase Firestore, Firebase Auth
- **AI:** Google Gemini Flash (crisis detection, OCR, volunteer matching, SITREPs, impact scoring)
- **Maps:** Google Maps JavaScript API
- **External Data:** GDACS (Global Disaster Alert and Coordination System)
- **Hosting:** Firebase Hosting (Google Cloud)

## 📦 Local Setup

```bash
git clone https://github.com/thejas-tech-hub/sevasphere-2026.git
cd sevasphere-2026
npm install
```

Create `.env` file:

REACT_APP_GEMINI_API_KEY=your_gemini_key
REACT_APP_MAPS_API_KEY=your_maps_key
REACT_APP_FIREBASE_API_KEY=your_firebase_key
REACT_APP_FIREBASE_AUTH_DOMAIN=sevasphere-2026.firebaseapp.com
REACT_APP_FIREBASE_PROJECT_ID=sevasphere-2026
REACT_APP_FIREBASE_STORAGE_BUCKET=sevasphere-2026.firebasestorage.app
REACT_APP_FIREBASE_MESSAGING_ID=your_messaging_id
REACT_APP_FIREBASE_APP_ID=your_app_id

```bash
npm start
```
