## 🌟 Problem Statement

Every year, 50+ million Indians are affected by natural disasters. Relief organizations lose **72 critical hours** coordinating volunteers — hours where lives are at stake. Existing volunteer platforms are **reactive** — they act after disaster peaks, not before.

## 🚀 Solution

SevaSphere uses **Gemini AI + real UN disaster data (GDACS)** to detect emerging crises across India and pre-mobilize verified volunteers — connecting NGOs, communities, and first responders in real-time.

**Key differentiator:** We act *before* the crisis peaks, not after.

## ✨ Core Features

| Feature | Description |
|---|---|
| 🌍 **Live Crisis Detection** | Pulls real disaster data from GDACS (UN system) + Gemini AI analysis |
| 🗺️ **Crisis Intelligence Map** | Google Maps with color-coded crisis pins, pulse zones, risk scores |
| 🤖 **AI Volunteer Matching** | Gemini matches volunteers by skill, location, and urgency |
| 📋 **AI Situation Report** | One-click government-ready SITREP generated by Gemini AI |
| 📊 **Impact Tracker** | AI-powered volunteer impact scoring with badges |
| 🏢 **NGO Command Center** | Real-time dashboard with crisis alerts and volunteer management |

## 🛠️ Tech Stack

- **Frontend:** React.js, TailwindCSS, Recharts
- **Backend:** Firebase Firestore, Firebase Auth
- **AI:** Google Gemini Flash (crisis detection, volunteer matching, situation reports)
- **Maps:** Google Maps JavaScript API
- **External Data:** GDACS (Global Disaster Alert and Coordination System)
- **Hosting:** Firebase Hosting (Google Cloud)

## 📦 Local Setup

```bash
git clone https://github.com/thejas-tech-hub/sevasphere-2026.git
cd sevasphere-2026
npm install
```

Create `.env` file:

REACT_APP_GEMINI_API_KEY=your_gemini_key
REACT_APP_MAPS_API_KEY=your_maps_key

```bash
npm start
```
