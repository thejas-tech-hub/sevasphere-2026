## 🌟 Problem Statement

Every year, 50+ million Indians are affected by natural disasters. Relief organizations lose **72 critical hours** coordinating volunteers. Existing platforms are reactive — they act after disaster peaks, not before. Field data from paper surveys gets lost, never digitized.

## 🚀 Solution

SevaSphere uses **Gemini AI + real UN disaster data (GDACS)** to detect emerging crises and pre-mobilize verified volunteers — connecting NGOs, communities, and first responders in real-time.

**Key differentiator:** We act *before* the crisis peaks, not after.

## ✨ Core Features

| Feature | Description |
|---|---|
| 🌍 **Live Crisis Detection** | Real disaster data from GDACS (UN system) + Gemini AI analysis |
| 📄 **Multimodal Field Reports** | NGOs photograph handwritten surveys — Gemini AI reads and digitizes instantly |
| 🗺️ **Crisis Intelligence Map** | Google Maps with crisis pins, pulse zones, risk scores |
| 🤖 **Hybrid AI Volunteer Matching** | JavaScript pre-filters top candidates by skill overlap, then Gemini AI ranks them with contextual scoring |
| 🔮 **Predictive Crisis Intelligence** | Gemini AI analyzes active crisis patterns to predict where volunteer support will be needed in the next 24–48 hours |
| 📋 **AI Situation Report** | One-click government-ready SITREP generated by Gemini AI |
| 📊 **Impact Tracker** | AI-powered volunteer impact scoring with badges |
| 🏢 **NGO Command Center** | Real-time dashboard with crisis alerts, volunteer management, and AI predictions |

## 🆕 Latest Updates (v2.1)

### 🤖 Hybrid AI Volunteer Matching
The volunteer matching system now uses a **two-stage hybrid pipeline**:
1. **JavaScript pre-filtering** — Eliminates volunteers with zero skill overlap; ranks remaining by match count and keeps the top 20. This prevents hitting Gemini's token limits with large volunteer pools.
2. **Gemini AI ranking** — The filtered candidates are sent to Gemini, which returns contextual scores (0–100) factoring in skill relevance, location proximity, and crisis urgency.
3. **Seamless fallback** — Local skill-match scores render instantly. Gemini scores arrive async and seamlessly replace them. If Gemini fails, the UI stays functional with local scores.

**Key functions:** `getSkillMatchScore()`, `matchVolunteers()`, `rankNeedsForVolunteer()`

### 🔮 Predictive Crisis Intelligence
A new `generatePredictions()` function analyzes active crisis patterns and uses Gemini AI to forecast where volunteer support will be needed in the **next 24–48 hours**. Predictions include:
- Probability score (60–95%)
- Timeframe (24h or 48h)
- Scientific basis (e.g., "downstream flooding from rising Brahmaputra levels")
- Specific recommended action for NGOs
- Results are cached for 30 minutes to conserve API quota

### 📊 NGO Dashboard — Predictions Tab
The NGO Command Center now has a third tab (**🔮 Predictions**) alongside Crisis Alerts and Your Needs:
- Amber/yellow color scheme (distinct from crisis red)
- Probability progress bars
- Timeframe badges (24 hours / 48 hours)
- AI reasoning and recommended action cards
- Disclaimer banner: *"These are AI predictions, not confirmed events"*
- A 5th stat card (🔮 3 Predictions) in the dashboard header
